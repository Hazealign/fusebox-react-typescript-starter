# fusebox-react-typescript-starter

## How to run for test?

1. Install dependencies: `npm install`
2. Run this command: `npm run serve`
3. Open your favorite browser, and go to `http://localhost:4444/`.

## Bundled Settings
 - TypeScript 2.6.x
 - Less for styling
 - TSLint (tslint-config-standard)
 - React 16
 - mobx & mobx-react
 - RxJS
